Getting Started

Install Design DNA and encode your taste in under five minutes.

Installation

Design DNA installs as slash commands in your Claude Code project. Run the installer and choose your scope.

npx design-dna

You'll be asked to choose between local (this project only) and global (all projects) installation.

  DESIGN DNA
  Design system enforcement for AI-assisted development

  Install scope:

  1) Local (this project only — ./.claude/commands/)
  2) Global (all projects — ~/.claude/commands/)

  > 1

  Installed 4 commands to .claude/commands/dc

  Next steps:
  1. Restart Claude Code to load the new commands
  2. Run /dc:init to capture your design DNA

Commands

Command What it does
/dc:init Scans your codebase, asks design questions, generates your DNA
/dc:check Audits code against your DNA, reports drift
/dc:update Updates your DNA from plain language, Figma, URLs, screenshots
/dc:help Lists all commands

What Gets Generated

After running /dc:init, your project will have a .design/ directory:

.design/
  DNA.md              # Your taste, encoded
  principles.md       # Design principles with examples
  components/         # Per-component specs
  patterns/           # Interaction and layout patterns
  lookbook/           # Golden reference implementations

Claude reads these files before generating any UI code. The DNA is the source of truth.

Workflow

1. Initialize

Run /dc:init in your project. Design DNA scans your existing code, asks you high-level questions about your design vision, and generates the DNA files.

2. Review

Open .design/DNA.md and read it. This is the seed for all future design work. If something is wrong in the seed, everything that grows from it will be wrong too.

3. Build

Write UI code with Claude as normal. Claude will read the DNA before generating any UI, ensuring consistency with your design system.

4. Check

Run /dc:check periodically to catch drift. Fix mechanical issues automatically. Review judgment calls yourself.

5. Evolve

When your taste changes, run /dc:update with a description of what's off. The DNA evolves with you.
